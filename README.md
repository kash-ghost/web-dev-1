Project- One-Page Portfolio Website
Overview-
This project is a one-page personal portfolio website built using semantic HTML5 and styled with external CSS. The site showcases key sections such as About, Projects, and Contact, with accessible, clean, and well-structured code, making it easy to maintain and enhance over time.

It is designed to follow best practices in web development, including semantic markup, responsive layout, accessibility improvements, and modular CSS styling.
